# Reprise du site complet

Le site utilise actuellement une page de transition dans `index.html` (orientation LinkedIn).

## Pour remettre le site complet en ligne

1. Ouvrir l'historique Git et retrouver le dernier `index.html` complet.
2. Restaurer ce fichier comme `index.html`.
3. Vérifier en local:
   - `python -m http.server 8000`
   - ouvrir `http://localhost:8000`
4. Contrôler rapidement les pages principales (`services.html`, `apropos.html`, `contact.html`).
5. Publier.

## Variante ultra rapide (si besoin)

- Si un backup local existe, remplacer simplement `index.html` actuel par la version complète sauvegardée.
